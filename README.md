# Roberto Hisao Kumagai | Full Stack Developer

Portafolio profesional creado con Vue 3, Vite, TailwindCSS y VueUse Motion.

## Stack

- Vue 3
- Vite
- TailwindCSS
- VueUse Motion
- Lucide Icons

## Comandos

```bash
npm install
npm run dev
npm run build
```

## Deploy en GitHub Pages

Este proyecto incluye GitHub Actions para publicar automaticamente el sitio cuando se suben cambios a la rama `main`.

En GitHub, ir a:

`Settings > Pages > Build and deployment > Source > GitHub Actions`

Luego hacer push a `main`.
